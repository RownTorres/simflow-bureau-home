# Simflow Bureau

Marketing site for Simflow Bureau, a Go HighLevel automation agency for service-based businesses.

## Stack

- Next.js 16 (App Router) + Turbopack
- Tailwind CSS 3
- Fraunces variable serif (Google Fonts)
- TypeScript

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy

Auto-deployed to Vercel on push to `main`.
